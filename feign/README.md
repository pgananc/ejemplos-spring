# migration-data-flyway
Migración de datos con flyway.
# Requisitos
* Mínimo JDK 11
* Maven 3.8.2

# Ejecución
* Ejecutar: mvn spring-boot:run

# EndPoints

* Get. Generación de migración.

	  http://localhost:8080/api/v1/employees/pageable-migration
