# feign
Utilización de feign para consumo de servicios.
# Requisitos
* Mínimo JDK 11
* Maven 3.8.2

# Ejecución
* Ejecutar: mvn spring-boot:run

# EndPoints

* Post. Generación de migración.

	 http://localhost:8081/api/v1/employees/Sistemas
	 
	{
	"firstName":"John",
	"lastName":"Rambo"
	}
