# consume-rest-api-authorization
Ejemplo de consumo de una rest-api con RestTemple con authorización.
# Requisitos
* Mínimo JDK 11
* Maven 3.8.2
# Ejecución.
* Ejecutar: mvn clean install
# EndPoints
* Consume (Post)
  * http://localhost:8080/api/v1/consume-rest-api-authorization
