# consume-rest-api
Ejemplo de consumo de una rest-api con RestTemple.
# Requisitos
* Mínimo JDK 11
* Maven 3.8.2
# Ejecución.
* Ejecutar: mvn clean install
# EndPoints
* Consume (Get)
  * http://localhost:8080/api/v1/consume-rest-api
